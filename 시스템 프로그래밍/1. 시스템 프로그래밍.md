# 시스템 프로그래밍



### 시스템 프로그래밍 과목 소개

- 고급 운영체제
- 운영체제 과목 : 이론 중심
- 시스템 프로그래밍 과목 : 실습 중심
- 실전 리눅스/운영체제 고급 활용
  - 리눅스 OS
    1. 인터넷 서비스 서버 환경 익히기
    2. 고급 운영체제 기능 직접 제어
    3. UNIX계열 OS 사용법 익히기



### 시스템 프로그래밍 선수 과목

- C 언어
- 자료 구조
- 컴퓨터 구조
- 운영체제



### 학습 방법

- 실제 실습

- 직접 작성해서 결과를 보면 이해도가 높아짐
- 조금씩 바꿔보면서 다른 결과를 보기
- 운영체제 과목 내용을 찾아보면서
- 끝까지 진행 후, 재반복
